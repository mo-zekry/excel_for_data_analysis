# Excel Keyboard Shortcuts Reference

## Navigation Shortcuts
| Action | Windows Shortcut | Mac Shortcut |
|--------|-----------------|--------------|
| Move to next cell | Tab | Tab |
| Move to previous cell | Shift+Tab | Shift+Tab |
| Move to beginning of row | Home | Fn+Left Arrow |
| Move to last cell in row with data | End, Right Arrow | Fn+Right Arrow |
| Move to first cell in worksheet | Ctrl+Home | Fn+Control+Left Arrow |
| Move to last cell with data | Ctrl+End | Fn+Control+Right Arrow |
| Move between worksheets | Ctrl+PgUp / Ctrl+PgDn | Fn+Option+Up/Down Arrow |
| Open Go To dialog | F5 or Ctrl+G | F5 or Command+G |

## Selection Shortcuts
| Action | Windows Shortcut | Mac Shortcut |
|--------|-----------------|--------------|
| Select entire column | Ctrl+Spacebar | Control+Spacebar |
| Select entire row | Shift+Spacebar | Shift+Spacebar |
| Select entire worksheet | Ctrl+A | Command+A |
| Extend selection | Shift+Arrow Keys | Shift+Arrow Keys |
| Select non-adjacent cells | Ctrl+Click | Command+Click |
| Select current region | Ctrl+Shift+* | Control+Shift+* |
| Select visible cells only | Alt+; | Command+Shift+Z |

## Lesson-Specific Shortcuts
These shortcuts are particularly relevant to specific lessons in this course:

### Lesson 1-2: Basics and Data Preparation
- Copy/Paste: Ctrl+C / Ctrl+V (Command+C / Command+V)
- Cut: Ctrl+X (Command+X)
- Undo: Ctrl+Z (Command+Z)
- Redo: Ctrl+Y (Command+Y)
- Hide rows/columns: Ctrl+9 / Ctrl+0 (Command+9 / Command+0)

### Lesson 3-4: Formulas and Functions
- Insert current date: Ctrl+; (Command+;)
- Insert function: Shift+F3
- Edit formula in cell: F2

### Lesson 5-7: Pivot Tables and Data Analysis
- Filter data: Ctrl+Shift+L (Command+Shift+F)
- Group data: Alt+Shift+Right Arrow (Option+Shift+Right Arrow)
- Data form: Alt+D, O

### Lesson 8: Practical Application
- Toggle formula view: Ctrl+` (Control+`)
- Hide/unhide worksheets: Ctrl+Shift+9 / Ctrl+Shift+0
